\begin{abstract}
本文围绕题目要求，建立可复现的数学模型，并通过程序求解、结构化验证报告和敏感性分析保证结果可靠。摘要应采用逐问题摘要写法，每一段对应一个子问题，明确说明模型、求解方法、关键结果和结论。

\problemabstract{问题一}{概述所用模型与主要求解流程。}{写入经 result registry 批准的关键结果。}
\problemabstract{问题二}{概述改进模型、约束处理或优化算法。}{写入经 verification report 通过的核心结论。}
\problemabstract{问题三}{概述扩展情形、参数变化或多因素耦合处理。}{写入敏感性分析与稳健性结论。}
\problemabstract{问题四}{概述最终方案、推广或调整策略。}{写入最终建议和适用边界。}

\keywords{数学建模；模型验证；敏感性分析；结果注册表；论文质量审计}
\end{abstract}
